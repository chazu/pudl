# B2.1 — Workspace Discovery and Loading

**Epic:** E2 — Per-Repo Workspace Support
**Blocked by:** nothing
**Blocks:** B2.2, B2.3, B2.4, B2.5

## Summary

Create a new `internal/workspace` package that discovers and loads per-repo workspace configuration by walking up from cwd looking for `.pudl/workspace.cue`.

## Background

- Currently `config.GetPudlDir()` always returns `~/.pudl/`
- `repo.Init()` creates a `.pudl/` directory in the repo but it's only a marker with no config
- The workspace package will be the central authority for "where am I running?" and merge per-repo config with global config
- CUE is already a dependency; workspace.cue can be parsed with the CUE Go API

## Implementation

### 1. New file: `internal/workspace/workspace.go`

```go
package workspace

import (
    "os"
    "path/filepath"
)

// Workspace represents a discovered per-repo workspace.
type Workspace struct {
    Root           string             // absolute path to repo root (parent of .pudl/)
    PudlDir        string             // absolute path to .pudl/ directory
    Name           string             // workspace name from workspace.cue
    SchemaPath     string             // .pudl/schema/ (may not exist)
    DefinitionsPath string            // .pudl/definitions/ (may not exist)
    ToolchainMappings []ToolchainOverride // per-repo toolchain overrides
}

// ToolchainOverride maps a schema prefix to a toolchain for this workspace.
type ToolchainOverride struct {
    Prefix    string
    Toolchain string
}

// Discover walks up from startDir looking for .pudl/workspace.cue.
// Returns nil if no workspace is found (global mode).
func Discover(startDir string) (*Workspace, error) {
    dir, err := filepath.Abs(startDir)
    if err != nil {
        return nil, err
    }

    for {
        candidate := filepath.Join(dir, ".pudl", "workspace.cue")
        if _, err := os.Stat(candidate); err == nil {
            return load(dir)
        }

        parent := filepath.Dir(dir)
        if parent == dir {
            return nil, nil // reached filesystem root, no workspace found
        }
        dir = parent
    }
}

// load parses workspace.cue and populates the Workspace struct.
func load(root string) (*Workspace, error)
```

### 2. Parsing workspace.cue

Use the CUE Go API to parse workspace.cue:

```go
import (
    "cuelang.org/go/cue"
    "cuelang.org/go/cue/cuecontext"
)

func load(root string) (*Workspace, error) {
    pudlDir := filepath.Join(root, ".pudl")
    cuePath := filepath.Join(pudlDir, "workspace.cue")

    data, err := os.ReadFile(cuePath)
    if err != nil {
        return nil, fmt.Errorf("reading workspace.cue: %w", err)
    }

    ctx := cuecontext.New()
    val := ctx.CompileBytes(data)
    if val.Err() != nil {
        return nil, fmt.Errorf("parsing workspace.cue: %w", val.Err())
    }

    ws := &Workspace{
        Root:            root,
        PudlDir:         pudlDir,
        SchemaPath:      filepath.Join(pudlDir, "schema"),
        DefinitionsPath: filepath.Join(pudlDir, "definitions"),
    }

    // Extract name (required)
    if name := val.LookupPath(cue.ParsePath("name")); name.Exists() {
        ws.Name, _ = name.String()
    } else {
        // Default to directory name
        ws.Name = filepath.Base(root)
    }

    // Extract toolchain_mappings (optional)
    if mappings := val.LookupPath(cue.ParsePath("toolchain_mappings")); mappings.Exists() {
        iter, _ := mappings.List()
        for iter.Next() {
            v := iter.Value()
            prefix, _ := v.LookupPath(cue.ParsePath("prefix")).String()
            toolchain, _ := v.LookupPath(cue.ParsePath("toolchain")).String()
            if prefix != "" && toolchain != "" {
                ws.ToolchainMappings = append(ws.ToolchainMappings, ToolchainOverride{
                    Prefix: prefix, Toolchain: toolchain,
                })
            }
        }
    }

    return ws, nil
}
```

### 3. New file: `internal/workspace/context.go`

Provide a workspace-aware context that commands can use:

```go
// Context holds the resolved workspace state for the current invocation.
type Context struct {
    // Workspace is non-nil when running inside a per-repo workspace.
    Workspace *Workspace

    // GlobalPudlDir is always ~/.pudl/
    GlobalPudlDir string

    // EffectiveOrigin is the workspace name (if in workspace) or "global"
    EffectiveOrigin string

    // SchemaSearchPaths is the ordered list of schema directories to search.
    // Per-repo first, then global, then embedded.
    SchemaSearchPaths []string

    // DefinitionSearchPaths is the ordered list of definition directories.
    DefinitionSearchPaths []string
}

// NewContext discovers the workspace and builds the resolved context.
func NewContext() (*Context, error) {
    cwd, err := os.Getwd()
    if err != nil {
        return nil, err
    }

    globalDir := config.GetPudlDir()
    ws, err := Discover(cwd)
    if err != nil {
        return nil, err
    }

    ctx := &Context{
        Workspace:     ws,
        GlobalPudlDir: globalDir,
    }

    if ws != nil {
        ctx.EffectiveOrigin = ws.Name
        ctx.SchemaSearchPaths = []string{ws.SchemaPath, filepath.Join(globalDir, "schema")}
        ctx.DefinitionSearchPaths = []string{ws.DefinitionsPath, filepath.Join(globalDir, "schema", "definitions")}
    } else {
        ctx.EffectiveOrigin = "global"
        ctx.SchemaSearchPaths = []string{filepath.Join(globalDir, "schema")}
        ctx.DefinitionSearchPaths = []string{filepath.Join(globalDir, "schema", "definitions")}
    }

    return ctx, nil
}
```

## workspace.cue Template

For use by `pudl repo init`:

```cue
// PUDL workspace configuration.
// This file marks the root of a per-repo PUDL workspace.

// Workspace name — used as the origin for catalog entries.
// Must be unique across all workspaces sharing the same ~/.pudl/ catalog.
name: "<directory-name>"

// Optional: override toolchain mappings for this workspace.
// These take priority over global config and built-in defaults.
// toolchain_mappings: [
//     {prefix: "myapp", toolchain: "shell"},
// ]
```

## Acceptance Criteria

- `workspace.Discover("/path/to/my-project/src/deep")` finds `.pudl/workspace.cue` in `/path/to/my-project/`
- `workspace.Discover("/tmp/no-workspace")` returns nil
- Workspace name, schema paths, definition paths, and toolchain mappings are correctly parsed
- `workspace.NewContext()` produces correct search path ordering (per-repo before global)
